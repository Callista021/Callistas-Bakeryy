# Callista's Bakery — Bootstrap 5 Multi-page Site

## Structure
- index.html — Home
- products.html — Product listing
- about.html — About & FAQ
- contact.html — Contact form (Bootstrap validation)
- banana-bread.html, choco-cake.html, croissant.html, muffin.html, blueberry-pie.html — Product detail pages
- bootstrap-tutorial.html — Short BS5 tutorial index
- style.css — Custom theme
- script.js — Front-end logic (products, toasts, search)

## How to run locally
1. Open the folder in VS Code.
2. Use Live Server extension or open `index.html` in a browser.
3. All pages are static files — no server required.
